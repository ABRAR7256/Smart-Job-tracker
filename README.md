# Smart-Job-tracker

A Python automation project that tracks job applications from Gmail.

## Features

- Connects to Gmail using IMAP
- Extracts job-related emails
- Identifies company names and statuses
- Saves data to CSV
- Creates simple visualizations

## Tech Stack

- Python
- Pandas
- Jupyter Notebook
- IMAP
- Matplotlib

## Setup

1. Clone the repository
2. Install dependencies
3. Create a .env file
4. Run the notebook
